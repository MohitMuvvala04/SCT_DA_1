# SCT_DA_1 - Excel Sales Dashboard

## About the Project
This project was completed as part of the Data Analytics Internship at SkillCraft Technology.

The objective was to analyze the Superstore Sales dataset and create an interactive sales dashboard using Microsoft Excel. The dataset was cleaned, organized, and analyzed using Pivot Tables and Pivot Charts to identify key sales insights.

## Dashboard Highlights
- Total Sales Overview
- Sales by Category Analysis
- Sales Trend Over Time
- Data Cleaning and Transformation
- Interactive Pivot Table Analysis

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts

## Files Included
- `Task1_Sales_Dashboard.xlsx` – Excel dashboard file
- `Sales_Dashboard.png` – Dashboard screenshot

## Key Insights
- Technology generated the highest sales among all categories.
- Sales showed an overall increasing trend over the years.
- The dashboard provides a quick overview of business performance and category-wise sales distribution.
